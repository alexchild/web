Using Jenkins to automate the build

Prerequistes: Have setup Tomcat7 and Jenkins. Have a Basic website to deploy

Currently you should be able to build the site manually by clicking the Build button.

The next step is configuring Jenkins to periodically re-build the site.

To do this click Configure, go to Build Triggers and select Build Periodically.

Here you can define a schedule, it is worth reading into how schedules are expressed to get a better understanding, but for the purpose of this guide you can use H/15 * * * * which builds the code every 15 miuntes.

Once you have clicked Apply and Save, sit back, relax and wait for Jenkins to build your site!
