Git/Jenkins Web Demo

Welcome to this site. This site is used to demo the use of git and Jenkins to build a website with version control and automated builds

This site can be replicated using the following steps:

  • Installing Tomcat7
  • Installing Jenkins
  • Configuring Jenkins
  • Setting Up a GitHub Account
  • Initiallising a Git Repo
  • Pushing a Repo to GitHub
  • Building a Basic site
  • Configuring Jenkins to build the site
  • Using Jenkins to automate the build
